# Web界面多列主键支持更新说明

## 更新概述

Web界面已更新以支持单列或多列组合主键配置。

## 界面变更

### 修改前
- 主键列输入框：数字输入框（type="number"）
- 只能输入单个数字

### 修改后
- 主键列输入框：文本输入框（type="text"）
- 支持单列或多列输入
- 添加了输入提示说明

## 使用方法

### 单列主键
在主键列输入框中输入单个数字：
```
0
```

### 多列主键
在主键列输入框中输入多个数字，用逗号分隔：
```
0,1
```
或
```
0,1,2
```

## 输入示例

### 示例1：单列主键（A列）
```
模型特征表主键列索引: 0
接口/灰度/从库特征表主键列索引: 0
```

### 示例2：双列主键（A列+B列）
```
模型特征表主键列索引: 0,1
接口/灰度/从库特征表主键列索引: 0,1
```

### 示例3：三列主键（A列+B列+C列）
```
模型特征表主键列索引: 0,1,2
接口/灰度/从库特征表主键列索引: 0,1,2
```

## 技术实现

### 前端解析函数

```javascript
/**
 * 解析主键列输入
 * 支持单列(数字)或多列(逗号分隔的数字)
 */
function parseKeyColumns(input) {
    const trimmed = input.trim();
    
    if (trimmed.includes(',')) {
        // 多列主键：分割并转换为数字数组
        const columns = trimmed.split(',')
            .map(s => s.trim())
            .filter(s => s !== '')
            .map(s => parseInt(s))
            .filter(n => !isNaN(n) && n >= 0);
        
        return columns.length > 0 ? columns : 0;
    } else {
        // 单列主键：直接转换为数字
        const num = parseInt(trimmed);
        return isNaN(num) || num < 0 ? 0 : num;
    }
}
```

### 格式化显示函数

```javascript
/**
 * 格式化主键列显示
 */
function formatKeyColumns(keyColumn) {
    if (Array.isArray(keyColumn)) {
        return keyColumn.join(',');
    }
    return String(keyColumn);
}
```

## 使用流程

### 1. 配置主键列
在主键列输入框中输入：
- 单列：`0`
- 多列：`0,1` 或 `0,1,2`

### 2. 调整特征起始列
如果使用多列主键，记得调整特征起始列索引：
- 双列主键 `[0,1]` → 特征起始列改为 `2`
- 三列主键 `[0,1,2]` → 特征起始列改为 `3`

### 3. 保存配置
点击"💾 保存配置"按钮保存当前配置

### 4. 执行对比
点击"▶️ 执行对比"按钮开始对比

## 常见问题

### Q: 输入格式错误会怎样？
A: 系统会自动过滤无效输入，保留有效的数字部分。

### Q: 可以输入空格吗？
A: 可以，系统会自动去除空格。例如 `0, 1, 2` 和 `0,1,2` 效果相同。

### Q: 如何知道输入是否正确？
A: 保存配置后，重新加载配置可以看到格式化后的值。

### Q: 旧配置还能用吗？
A: 可以，系统会自动识别并正确显示旧配置。
