## 🚀 使用方法

### 启动服务
```bash
cd MyDataCheck
./start_web.sh
```

### 访问页面
```
http://localhost:5001
```

### 使用步骤
1. 点击"📊 数据对比" Tab
2. 上传文件1（CSV或XLSX）
3. 上传文件2（CSV或XLSX）
4. 配置主键列索引和特征起始列索引
5. 点击"▶️ 执行对比"
6. 等待执行完成
7. 查看生成的报告文件

---

## 📝 注意事项

1. **文件格式**：只支持 CSV 和 XLSX 格式
2. **文件大小**：单个文件最大 1GB
3. **主键列**：主键列的值必须唯一
4. **列索引**：列索引从 0 开始（A列=0，B列=1，...）
5. **编码**：CSV文件会自动检测编码（UTF-8、GBK等）
6. **依赖**：需要安装 pandas 库（用于读取XLSX文件）
