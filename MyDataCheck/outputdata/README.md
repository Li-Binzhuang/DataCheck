# 输出数据目录说明

本目录用于存储数据对比和线上灰度落数对比的输出文件。

## 目录结构

```
outputdata/
├── api_comparison/       # 接口数据对比的输出文件
│   ├── {前缀}_{文件名}_{时间戳}_api_data.csv
│   └── {前缀}_{文件名}_{时间戳}_compare.csv
│
└── online_comparison/    # 线上灰度落数对比的输出文件
    ├── {前缀}_{文件名}_{时间戳}_解析后.csv
    └── {前缀}_{文件名}_{时间戳}_对比结果_*.csv
```

## 文件说明

### 接口数据对比输出文件

- `*_api_data.csv`: 从接口获取的数据文件
- `*_compare.csv`: 对比结果文件
- `*_compare_analysis_report.csv`: 分析报告文件
- `*_compare_feature_stats.csv`: 特征统计文件

### 线上灰度落数对比输出文件

- `*_解析后.csv`: JSON解析后的CSV文件
- `*_对比结果_差异特征汇总.csv`: 差异特征汇总文件
- `*_对比结果_差异数据明细.csv`: 差异数据明细文件
- `*_对比结果_特征统计.csv`: 特征统计文件

## 注意事项

- 所有输出文件都会自动保存到此目录
- 文件命名格式：`{前缀}_{原文件名}_{时间戳}_{类型}.csv`
- 时间戳格式：`MMDDHHmm`（月日时分）
