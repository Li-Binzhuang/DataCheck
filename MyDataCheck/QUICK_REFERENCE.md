# 小数处理工具 - 快速参考

## 🎯 访问方式

### 新方式（推荐）
1. 打开Web界面
2. 点击左侧菜单"🔢 小数处理工具"
3. 直接进入小数处理工具页面

### 位置
左侧菜单栏，位于"📊 csv数据对比"和"📦 PKL文件解析"之间

## 📋 功能清单

### 文件输入
- ✅ 上传文件模式
- ✅ 服务器路径模式（大文件推荐）

### 处理方式（5种）
1. 不处理
2. 四舍五入
3. 双精度四舍五入
4. 截取
5. 向上取整

### 对比方式（4种）
1. 精确对比
2. 容差对比
3. 最后一位差1不计异常
4. 最后一位差2不计异常

### 配置项
- 源列（需要处理的列）
- 参考列（提供小数位数的列）
- 输出文件前缀
- 输出全量数据选项

## 🚀 快速使用

```
1. 点击"小数处理工具"菜单
2. 上传CSV文件或输入服务器路径
3. 输入源列名称（如：API值）
4. 输入参考列名称（如：CSV值）
5. 选择处理方式（默认：四舍五入）
6. 选择对比方式（默认：精确对比）
7. 点击"执行处理"
8. 查看输出结果
```

## 📊 输出文件

### 差异报告
- 文件名：`{前缀}_差异明细_{时间戳}.csv`
- 内容：只包含有差异的记录

### 全量数据（可选）
- 文件名：`{前缀}_全量数据_{时间戳}.csv`
- 内容：包含所有记录的处理结果

## 💡 使用技巧

### 大文件处理
- 使用"服务器路径"模式
- 文件放在 `inputdata/data_comparison/` 目录
- 输入相对路径或绝对路径

### 精度控制
- 四舍五入：标准四舍五入
- 双精度四舍五入：减少浮点精度误差
- 截取：直接截断小数
- 向上取整：向上取整到指定位数

### 对比策略
- 精确对比：差异为0才不计为差异
- 容差对比：差异在容差范围内不计为差异
- 最后一位差1：最后一位允许±1误差
- 最后一位差2：最后一位允许±2误差
