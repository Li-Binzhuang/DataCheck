## 统一的输出控制逻辑

现在所有板块（除 BOSS 板块外）都采用统一的双开关设计：

```
┌─────────────────────────────────────────────────────────┐
│                    WRITE_OUTPUTS                        │
│              (总开关，控制是否输出文件)                    │
└─────────────────────────────────────────────────────────┘
                          │
                ┌─────────┴─────────┐
                │                   │
            True                 False
                │                   │
                ▼                   ▼
        ┌───────────────┐    不输出任何文件
        │ WRITE_FLAT_CSV│
        │(明细表开关)    │
        └───────────────┘
                │
        ┌───────┴───────┐
        │               │
     True            False
        │               │
        ▼               ▼
   输出明细表      跳过明细表
        │               │
        └───────┬───────┘
                │
                ▼
         总是输出特征表
```

## 默认配置

### 第一、二、三板块

```python
WRITE_OUTPUTS = True      # 启用输出
WRITE_FLAT_CSV = False    # 不输出明细表（默认）
```

**输出结果**：
- ✅ 只输出特征表
- ❌ 跳过明细表

**优点**：
- 节省磁盘空间（尤其是第二板块，明细表约 200MB）
- 输出速度快
- 满足大多数使用场景

## 文件大小对比

| 板块 | 明细表大小 | 特征表大小 | 节省空间（默认配置） |
|------|-----------|-----------|-------------------|
| 第一板块 | 10-50 MB | 5-10 MB | 10-50 MB |
| 第二板块 | 100-200 MB | 50-100 MB | 100-200 MB |
| 第三板块 | 5-20 MB | 2-5 MB | 5-20 MB |
