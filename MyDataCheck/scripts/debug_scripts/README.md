# 调试脚本目录

本目录包含项目的各种调试和诊断脚本。

## 📋 文件说明

### 调试脚本
- `debug_merge.py` - 合并调试脚本
- `diagnose_csv.py` - CSV 诊断脚本
- `reorganize_project.py` - 项目重组脚本

### 调试数据
- `debug_vertical_output.csv` - 垂直输出调试数据

## 🚀 使用方法

运行调试脚本：
```bash
python debug_merge.py
python diagnose_csv.py
# 等等...
```

## 📝 注意事项

- 这些是开发调试用的脚本
- 用于诊断和解决问题
- 不应在生产环境中使用
