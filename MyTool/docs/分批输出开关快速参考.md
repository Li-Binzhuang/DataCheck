# 分批输出开关快速参考

## 配置项

```python
ENABLE_BATCH_OUTPUT = True  # True: 分批输出，False: 单文件输出
```

## 快速配置

### 分批输出模式

```python
ENABLE_BATCH_OUTPUT = True
BATCH_SIZE = 2000  # 每批2000条
```

**输出结果**：
- `cdc1_features_batch001_1-2000.csv`
- `cdc1_features_batch002_2001-4000.csv`
- ...

### 单文件输出模式

```python
ENABLE_BATCH_OUTPUT = False
```

**输出结果**：
- `cdc1_features.csv`（全量数据）

## 四个板块默认配置

| 板块 | ENABLE_BATCH_OUTPUT | BATCH_SIZE | 输出文件前缀 |
|------|---------------------|------------|--------------|
| 第一板块 | True | 2100 | cdc1_features |
| 第二板块 | True | 200 | cdc2_features |
| 第三板块 | True | 6150 | cdc3_features |
| BOSS板块 | True | 6150 | cdcboss_features |

## 常见场景

### 场景1：只要单个全量文件

```python
ENABLE_BATCH_OUTPUT = False
WRITE_FULL_CSV = False  # 不需要额外的全量文件
```

### 场景2：只要分批文件

```python
ENABLE_BATCH_OUTPUT = True
BATCH_SIZE = 2000
WRITE_FULL_CSV = False
```

### 场景3：分批文件 + 全量文件都要

```python
ENABLE_BATCH_OUTPUT = True
BATCH_SIZE = 2000
WRITE_FULL_CSV = True
```

## 修改位置

在每个衍生脚本的输出配置区域（通常在脚本末尾）：

```python
# 输出配置区域
WRITE_OUTPUTS = True
ENABLE_BATCH_OUTPUT = True  # ← 修改这里
BATCH_SIZE = 2000
WRITE_FULL_CSV = True
```
