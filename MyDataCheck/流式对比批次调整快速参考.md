# 流式对比批次调整 - 快速参考

## 📋 调整内容

| 项目 | 修改前 | 修改后 |
|------|--------|--------|
| 批次大小 | 1000 条/批 | **50 条/批** |
| 进度显示 | 每 10 批显示一次 | **每批都显示** |
| 进度频率 | 每 10,000 条显示一次 | **每 50 条显示一次** |

## 🔧 修改的文件

1. `MyDataCheck/api_comparison/job/streaming_comparator.py`
   - 第 48 行: `batch_size: int = 50`
   - 第 332-334 行: 移除进度显示条件判断

2. `MyDataCheck/api_comparison/job/process_executor.py`
   - 第 200 行: `batch_size: int = 50`
   - 第 421 行: `batch_size=50`

## ⚙️ 自定义批次大小

修改 `streaming_comparator.py` 第 48 行：
```python
batch_size: int = 50,  # 改为你想要的值
```

**推荐值**:
- 小数据集（< 1000）: `50`
- 中数据集（1000-5000）: `50-100`
- 大数据集（> 5000）: `100-200`
