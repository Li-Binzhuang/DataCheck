## 📝 注释规范

### 模块文档字符串
```python
"""
模块名称

功能说明:
    - 功能点1
    - 功能点2

作者: MyDataCheck Team
创建时间: YYYY-MM
最后更新: YYYY-MM-DD
"""
```

### 函数文档字符串
```python
def function_name(param1, param2):
    """
    函数简短描述
    
    Args:
        param1 (type): 参数说明
        param2 (type): 参数说明
    
    Returns:
        type: 返回值说明
    
    示例:
        >>> function_name(value1, value2)
        result
    """
```
