# 接口对比配置路径修复

**修复时间**：2026-01-27  
**问题**：保存配置提示路径错误（路径重复）  
**状态**：✅ 已修复

---

## 问题描述

保存接口对比配置时报错：
```
保存失败: [Errno 2] No such file or directory: 
'/Users/.../MyDataCheck/api_comparison/api_comparison/config.json'
```

路径中`api_comparison`重复了两次。

---

## 问题原因

**路径拼接错误**

### 问题链
1. 修复接口对比卡住问题时，将`script_dir`改为：
   ```python
   script_dir = os.path.join(SCRIPT_DIR, "api_comparison")
   # 结果：MyDataCheck/api_comparison/
   ```

2. 但保存/加载配置时，又拼接了一次`api_comparison`：
   ```python
   config_file_path = os.path.join(script_dir, "api_comparison", "config.json")
   # 结果：MyDataCheck/api_comparison/api_comparison/config.json  ❌
   ```

3. 导致路径错误，文件无法保存

---

## 修复方案

### 修改文件
`web/routes/api_routes.py`

### 修改内容

#### 1. 保存配置路由

**修复前**
```python
@api_bp.route('/api/config/save', methods=['POST'])
def save_config():
    config_file_path = os.path.join(script_dir, "api_comparison", "config.json")  # ❌
    with open(config_file_path, 'w', encoding='utf-8') as f:
        json.dump(config_data, f, ensure_ascii=False, indent=2)
```

**修复后**
```python
@api_bp.route('/api/config/save', methods=['POST'])
def save_config():
    config_file_path = os.path.join(script_dir, "config.json")  # ✅
    with open(config_file_path, 'w', encoding='utf-8') as f:
        json.dump(config_data, f, ensure_ascii=False, indent=2)
```

#### 2. 加载配置路由

**修复前**
```python
@api_bp.route('/api/config/load', methods=['GET'])
def load_config():
    config_file_path = os.path.join(script_dir, "api_comparison", "config.json")  # ❌
    if os.path.exists(config_file_path):
        with open(config_file_path, 'r', encoding='utf-8') as f:
            config_data = json.load(f)
```

**修复后**
```python
@api_bp.route('/api/config/load', methods=['GET'])
def load_config():
    config_file_path = os.path.join(script_dir, "config.json")  # ✅
    if os.path.exists(config_file_path):
        with open(config_file_path, 'r', encoding='utf-8') as f:
            config_data = json.load(f)
```
