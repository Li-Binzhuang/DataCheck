- 使用缓冲区批量写入，默认 batch_size=5000

**核心特性**:
```python
class CSVBatchWriter:
    def __init__(self, file_path, headers, batch_size=5000):
        self.buffer = []  # 数据缓冲区
        self.batch_size = batch_size
    
    def write_row(self, row):
        self.buffer.append(row)
        if len(self.buffer) >= self.batch_size:
            self.flush()  # 批量写入
    
    def flush(self):
        self.writer.writerows(self.buffer)  # 一次性写入多行
        self.buffer.clear()
```

## 技术细节

### 批量写入原理

**问题**: 逐行写入导致频繁的I/O操作
```python
for row in data:
    writer.writerow(row)  # 每次都触发I/O
```

**解决**: 累积多行后一次性写入
```python
buffer = []
for row in data:
    buffer.append(row)
    if len(buffer) >= 5000:
        writer.writerows(buffer)  # 批量写入5000行
        buffer.clear()
```

**效果**:
- I/O次数减少 5000倍
- 系统调用次数大幅减少
- 写入速度提升 5-20倍

### 缓冲区优化

**增加文件缓冲区大小**:
```python
open(file_path, "w", encoding="utf-8", newline="", buffering=8192*8)
```

**效果**:
- 默认缓冲区：8KB
- 优化后缓冲区：64KB
- 减少系统调用次数
- 额外提升 10-30% 性能

## 使用说明

### 在新代码中使用批量写入

```python
from common.csv_tool import CSVBatchWriter

# 使用批量写入器
with CSVBatchWriter("output.csv", headers, batch_size=5000) as writer:
    for row in data_generator():
        writer.write_row(row)
```

### 批量添加多行

```python
with CSVBatchWriter("output.csv", headers) as writer:
    # 单行添加
    writer.write_row(row1)
    
    # 批量添加
    writer.write_rows([row2, row3, row4])
```

### 自动刷新

```python
# 不需要手动调用 flush()
# 退出上下文管理器时自动刷新剩余数据
with CSVBatchWriter("output.csv", headers) as writer:
    for row in data:
        writer.write_row(row)
# 这里自动调用 flush() 写入剩余数据
```
