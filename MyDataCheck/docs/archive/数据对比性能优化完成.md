## 技术细节

### 核心优化

**1. 字典索引（最关键）**
```python
# 优化前：O(n²)
for row_api in rows_api:
    for row_sql in rows_sql:  # 嵌套循环
        if match: break

# 优化后：O(n)
sql_index = {key: row for row in rows_sql}  # 一次性构建
for row_api in rows_api:
    row_sql = sql_index.get(key)  # O(1)查找
```

**2. 预先映射**
```python
# 一次性构建特征映射
feature_mapping = {
    feature: (api_idx, sql_idx)
    for feature in features
}
```

**3. 批量输出**
```python
# 动态调整输出频率
progress_interval = max(100, len(rows) // 20)  # 每5%
```
