# 场景1_接口数据对比 - 目录结构说明

## 目录结构

```
场景1_接口数据对比/
├── 执行对比流程.py          # 主执行脚本（推荐使用）
├── 使用说明.md              # 使用说明文档
├── 脚本功能分析.md          # 脚本功能分析文档
├── 目录结构说明.md          # 本文件
├── json/                    # JSON配置文件文件夹
│   ├── __init__.py
│   ├── 执行配置.json        # JSON配置文件
│   ├── 配置模板.json        # 配置模板（含详细说明）
│   ├── 列索引配置.json      # 列索引配置文件（示例）
│   └── 列索引配置_*.json    # 自动生成的列索引配置文件
├── job/                     # 功能模块文件夹
│   ├── __init__.py
│   ├── 检测列索引.py        # 检测列索引工具模块
│   ├── 获取接口数据.py      # 获取接口数据功能模块
│   ├── 对比接口数据.py      # 对比数据功能模块
│   └── 接口校验.py          # 旧版脚本（冗余，可删除）
└── [输出文件]               # 执行后生成的CSV/Excel文件
```

## 脚本说明

### 主脚本

**执行对比流程.py**
- 位置：场景1_接口数据对比/（根目录）
- 功能：主执行脚本，整合所有功能
- 调用：job文件夹中的功能模块
- 使用：直接运行此脚本

### 功能模块（job文件夹）

**job/检测列索引.py**
- 功能：自动检测CSV文件中cust_no和时间字段的列索引
- 被调用：执行对比流程.py
- 可独立运行：是

**job/获取接口数据.py**
- 功能：读取CSV文件，请求接口，输出接口全部数据
- 被调用：执行对比流程.py
- 可独立运行：是

**job/对比接口数据.py**
- 功能：对比原始数据和接口返回的数据
- 被调用：执行对比流程.py
- 可独立运行：是

**job/接口校验.py**
- 状态：⚠️ 旧版脚本，冗余
- 建议：删除或保留作为历史参考

## 导入路径说明

### 执行对比流程.py
- 导入公共工具：`../公共工具/`（上一级目录）
- 导入job模块：`job/`（当前目录下的job文件夹）

### job文件夹中的脚本
- 导入公共工具：`../../公共工具/`（上两级目录）
- 原因：job文件夹在场景1_接口数据对比下，需要回到数据校对目录才能访问公共工具

## 使用方式

### 方式1：使用主脚本（推荐）

```bash
python 执行对比流程.py
```

### 方式2：独立运行功能模块

如果需要单独运行某个功能模块：

```bash
# 检测列索引
python job/检测列索引.py

# 获取接口数据
python job/获取接口数据.py

# 对比接口数据
python job/对比接口数据.py
```

## 注意事项

1. **配置文件位置**：`执行配置.json` 应该在 `json/` 文件夹下
2. **输入文件位置**：CSV输入文件应该在场景1_接口数据对比目录下（或使用绝对路径）
3. **输出文件位置**：所有输出文件会生成在场景1_接口数据对比目录下
4. **列索引配置**：自动生成的列索引配置文件会保存在 `json/` 文件夹下

## 文件移动说明

所有功能模块脚本已移动到 `job/` 文件夹，主脚本 `执行对比流程.py` 保留在根目录，便于直接运行。
