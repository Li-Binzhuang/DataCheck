# 小数处理工具快速参考

## 使用流程

```
1. 上传CSV文件
   ↓
2. 指定源列（需要处理的列）
   ↓
3. 指定参考列（提供小数位数的列）
   ↓
4. 选择处理方式（四舍五入/截取/向上取整等）
   ↓
5. 选择对比方式（精确/容差/最后一位差1等）
   ↓
6. 执行处理
   ↓
7. 下载输出文件
```

## 快速配置

### 典型场景：处理API值
```
源列：API值
参考列：CSV值
处理方式：四舍五入
对比方式：精确对比
```

### 输出结果
```
原始列：CSV值, API值
新增列：API值-四舍五入, 差异值
```

## 处理方式对比

| 处理方式 | 示例输入 | 参考列 | 输出结果 |
|---------|---------|--------|---------|
| 不处理 | 100.1234 | 100.123 (3位) | 100.1234 |
| 四舍五入 | 100.1234 | 100.123 (3位) | 100.123 |
| 双精度四舍五入 | 100.1234 | 100.123 (3位) | 100.123 |
| 截取 | 100.1234 | 100.123 (3位) | 100.123 |
| 向上取整 | 100.1234 | 100.123 (3位) | 100.124 |

## 对比方式说明

| 对比方式 | 说明 | 示例 |
|---------|------|------|
| 精确对比 | 差异=0才不计 | 0.000 ✅, 0.001 ❌ |
| 容差对比(0.01) | \|差异\|≤0.01不计 | 0.009 ✅, 0.011 ❌ |
| 最后一位差1 | 最后一位±1不计 | 0.001 ✅, 0.002 ❌ (3位小数) |
| 最后一位差2 | 最后一位±2不计 | 0.002 ✅, 0.003 ❌ (3位小数) |

## 输出文件

### 完整处理结果（可选）
- 文件名：`decimal_processed_full_MMDDHHMM.csv`
- 包含：所有记录 + 处理后的列 + 差异值列

### 差异记录
- 文件名：`decimal_processed_diff_MMDDHHMM.csv`
- 包含：只有差异的记录

## 常用列名

### 数据对比输出文件
- 源列：`API值`
- 参考列：`CSV值`

### 接口对比输出文件
- 源列：`接口/灰度/从库值`
- 参考列：`模型特征样本值`

## 注意事项

✅ 列名必须完全匹配（区分大小写）  
✅ 处理后的列会插入到源列后面  
✅ 差异值 = 参考列 - 处理后的列  
✅ 只输出有差异的记录到差异文件  

## 快速示例

### 输入文件
```csv
特征名,CSV值,API值
feat1,100.123,100.1234
feat2,200.456,200.4567
```

### 配置
- 源列：`API值`
- 参考列：`CSV值`
- 处理方式：四舍五入
- 对比方式：精确对比

### 输出（完整）
```csv
特征名,CSV值,API值,API值-四舍五入,差异值
feat1,100.123,100.1234,100.123,0.0
feat2,200.456,200.4567,200.457,0.001
```

### 输出（差异）
```csv
特征名,CSV值,API值,API值-四舍五入,差异值
feat2,200.456,200.4567,200.457,0.001
```
