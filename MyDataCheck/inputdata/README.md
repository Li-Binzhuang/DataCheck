# 输入数据目录说明

本目录用于存储Web界面上传的输入文件。

## 目录结构

```
inputdata/
├── api_comparison/       # 接口数据对比的输入文件
│   └── *.csv             # 上传的CSV文件
│
└── online_comparison/     # 线上灰度落数对比的输入文件
    ├── *.csv             # 上传的线上文件
    └── *.csv             # 上传的离线文件
```

## 文件说明

### 接口数据对比输入文件

- 通过Web界面上传的CSV文件会保存到此目录
- 文件用于与接口返回的数据进行对比

### 线上灰度落数对比输入文件

- **线上文件**: 包含JSON列的线上数据文件
- **离线文件**: 离线数据文件

## 注意事项

- 所有通过Web界面上传的文件都会自动保存到此目录
- 文件命名会使用安全文件名（secure_filename）
- 如果文件已存在，新上传的文件会覆盖旧文件
