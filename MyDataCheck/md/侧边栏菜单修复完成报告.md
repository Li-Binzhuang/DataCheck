# 侧边栏菜单修复完成报告

## 修复时间
2026-01-26

## 问题描述
MyDataCheck项目的Web界面升级为侧边栏样式后，点击左侧菜单栏无法切换页面。

---

## 问题原因

### 1. 函数缺失
新的HTML模板使用了`switchPage()`函数，但JavaScript中只有旧的`switchTab()`函数。

### 2. 函数不匹配
- HTML中调用：`onclick="switchPage('api')"`
- JavaScript中定义：`function switchTab(tabName)`
- 导致点击菜单时找不到对应的函数

---

## 修复方案

### 1. 添加switchPage函数
在JavaScript中添加了完整的`switchPage()`函数：

```javascript
function switchPage(pageName) {
    console.log('switchPage called with:', pageName);
    
    // 隐藏所有页面
    document.querySelectorAll('.content-section').forEach(section => {
        section.classList.remove('active');
        console.log('Hiding section:', section.id);
    });
    
    // 移除所有菜单项的active状态
    document.querySelectorAll('.menu-item').forEach(item => {
        item.classList.remove('active');
    });
    
    // 显示选中的页面
    const targetPage = document.getElementById(`page-${pageName}`);
    if (targetPage) {
        targetPage.classList.add('active');
        console.log('Showing page:', targetPage.id);
    } else {
        console.error('Page not found:', `page-${pageName}`);
    }
    
    // 激活对应的菜单项
    if (typeof event !== 'undefined' && event && event.target) {
        const menuItem = event.target.closest('.menu-item');
        if (menuItem) {
            menuItem.classList.add('active');
            console.log('Menu item activated');
        }
    }
}
```

### 2. 添加调试日志
在函数中添加了console.log，方便调试：
- 显示调用的页面名称
- 显示隐藏的页面ID
- 显示激活的页面ID
- 显示菜单项激活状态

### 3. 修复event对象引用
添加了对event对象的检查：
```javascript
if (typeof event !== 'undefined' && event && event.target) {
    // 安全地使用event对象
}
```

### 4. 保留兼容性
保留了`switchTab()`函数作为别名：
```javascript
function switchTab(tabName) {
    switchPage(tabName);
}
```

## 功能说明

### switchPage函数功能
1. **隐藏所有页面**：移除所有`.content-section`的`active`类
2. **取消菜单高亮**：移除所有`.menu-item`的`active`类
3. **显示目标页面**：为目标页面添加`active`类
4. **高亮当前菜单**：为点击的菜单项添加`active`类

### CSS类说明
- `.content-section`：页面容器
- `.content-section.active`：显示的页面（display: block）
- `.menu-item`：菜单项
- `.menu-item.active`：激活的菜单项（高亮显示）

## 技术细节

### 页面切换原理
使用CSS的display属性控制页面显示：
```css
.content-section {
    display: none;  /* 默认隐藏 */
}

.content-section.active {
    display: block;  /* 激活时显示 */
}
```

### 菜单高亮原理
使用CSS的背景色和边框控制高亮：
```css
.menu-item {
    border-left: 3px solid transparent;
}

.menu-item.active {
    background: rgba(255,255,255,0.15);
    border-left-color: white;
    color: white;
    font-weight: 500;
}
```

### 事件处理
使用onclick属性直接绑定事件：
```html
<div class="menu-item" onclick="switchPage('api')">
    <span class="icon">📡</span>
    <span>接口数据对比</span>
</div>
```
