# 数据对比输出目录

## 说明
此目录用于存放数据对比的结果文件。

## 输出文件说明

### 1. 差异特征汇总
文件名：`{prefix}_{timestamp}_差异特征汇总.csv`

包含内容：
- 特征名
- 差异数量
- 总对比次数
- 差异占比(%)

### 2. 差异数据明细
文件名：`{prefix}_{timestamp}_差异数据明细.csv`

包含内容：
- 主键值
- cust_no
- 特征名
- 接口文件值
- Sql文件值

### 3. 特征统计
文件名：`{prefix}_{timestamp}_特征统计.csv`

包含内容：
- 特征名
- 总对比次数
- 一致数量
- 差异数量
- 一致率(%)

### 4. 全量数据合并
文件名：`{prefix}_{timestamp}_全量数据合并.csv`

包含内容：
- 主键值
- 数据来源（Sql文件/接口文件）
- 对比结果（一致/有差异）
- 所有特征列的值

### 5. 仅在接口文件中的数据
文件名：`{prefix}_{timestamp}_仅在接口文件中的数据.csv`

包含内容：
- 接口文件的完整记录（在Sql文件中找不到匹配的记录）

### 6. 仅在Sql文件中的数据
文件名：`{prefix}_{timestamp}_仅在Sql文件中的数据.csv`

包含内容：
- Sql文件的完整记录（在接口文件中找不到匹配的记录）

## 文件命名规则
- `{prefix}`: 配置文件中指定的输出前缀
- `{timestamp}`: 执行时间戳（格式：MMDDHHmm）

## 示例
```
outputdata/data_comparison/
├── compare_01261430_差异特征汇总.csv
├── compare_01261430_差异数据明细.csv
├── compare_01261430_特征统计.csv
├── compare_01261430_全量数据合并.csv
├── compare_01261430_仅在接口文件中的数据.csv
├── compare_01261430_仅在Sql文件中的数据.csv
└── README.md
```

## 注意事项
- 文件使用UTF-8-BOM编码，可直接用Excel打开
- 建议定期清理旧的输出文件
- 大文件可能占用较多磁盘空间
