# Scripts 目录说明

本目录包含项目的辅助脚本工具。

## 📁 目录结构

```
scripts/
├── setup/          # 环境设置脚本
└── archive/        # 归档脚本（历史脚本）
```

## 🔧 setup/ - 环境设置脚本

用于首次安装和环境配置的脚本。

### setup_python312.sh
**用途**: 一键设置Python 3.12环境

**功能**:
- 检查并安装Python 3.12
- 创建虚拟环境
- 安装项目依赖

**使用**:
```bash
./scripts/setup/setup_python312.sh
```

### install_pandas.sh
**用途**: 安装pandas依赖（处理Python 3.15兼容性问题）

**功能**:
- 检查Python版本
- 安装pandas
- 处理版本兼容性

**使用**:
```bash
./scripts/setup/install_pandas.sh
```

## 📦 archive/ - 归档脚本

历史脚本，已完成一次性任务，保留作为参考。

**状态**: ✅ 已完成 - 项目已使用5001端口

### cleanup_redundant_files.sh
**用途**: 清理项目冗余文件（已完成）

**状态**: ✅ 已完成 - 项目已清理完成

### organize_files.sh
**用途**: 整理项目文件（部分文档）（已完成）

**状态**: ✅ 已完成 - 文档已整理完成

### organize_all_files.sh
**用途**: 完整整理项目文件（所有文档）（已完成）

**状态**: ✅ 已完成 - 所有文档已整理完成

## 📝 说明

- **setup/** 目录的脚本可以在需要时重复使用
- **archive/** 目录的脚本仅作为历史参考，不建议再次执行
- 如需执行归档脚本，请先检查当前项目状态，避免重复操作
