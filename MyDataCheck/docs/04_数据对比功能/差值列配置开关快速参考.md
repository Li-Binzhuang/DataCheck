# 差值列配置开关 - 快速参考

## 一句话说明

控制接口数据对比报告中是否输出"差值"列（CSV值 - API值）。

## 配置位置

接口数据对比 → 场景配置 → `☐ 计算差值（CSV值 - API值）`

## 默认值

❌ **未勾选**（`false`）- 不计算差值

## 快速使用

### 启用差值计算
```
1. 打开接口数据对比页面
2. 展开场景配置
3. 勾选 ☑ 计算差值（CSV值 - API值）
4. 保存配置
5. 执行对比
```

### 输出效果

**启用时**:
```csv
特征名,applyId,request_time,CSV值,API值,差值
feature1,123,2025-01-01T12:00:00.000,100.5,100.0,0.5
```

**禁用时**:
```csv
特征名,applyId,request_time,CSV值,API值
feature1,123,2025-01-01T12:00:00.000,100.5,100.0
```

## 差值计算

- **公式**: `差值 = CSV值 - API值`
- **精度**: 6位小数
- **异常**: 非数值显示为空

## 适用场景

✅ 需要量化偏差  
✅ 需要分析差值分布  
✅ 需要精确对比数值  

❌ 只关心是否匹配  
❌ 特征值非数值类型  
❌ 需要减小文件大小  

## JSON配置

```json
{
  "scenarios": [{
    "calculate_difference": true
  }]
}
```

## 注意事项

- 只影响 `_analysis_report.csv`
- 不影响 `_feature_stats.csv`
- 不影响对比逻辑
- 需重新执行对比生效
