# 数据对比功能 - 快速参考

## 🚀 快速开始

### 1. 启动服务
```bash
cd MyDataCheck
./start_web.sh
```

### 2. 访问页面
打开浏览器：`http://localhost:5001`

### 3. 点击"📊 数据对比" Tab

### 4. 上传两个文件（CSV或XLSX）

### 5. 配置参数
- 主键列索引（A列=0）
- 特征起始列索引
- 转换特征值为数值（默认勾选）

### 6. 点击"▶️ 执行对比"

---

## 📊 输出文件

生成在 `outputdata/compare/` 目录：

1. **差异特征汇总.csv** - 哪些特征有差异
2. **差异数据明细.csv** - 具体差异记录
3. **特征统计.csv** - 匹配率统计
4. **全量数据合并.csv** - 完整对比结果

## 🔍 常见配置

### 配置1：标准对比（启用数值转换）
```
文件1主键列索引: 0 (apply_id)
文件2主键列索引: 0 (apply_id)
文件1特征起始列: 2 (从第3列开始)
文件2特征起始列: 2 (从第3列开始)
转换特征值为数值: 勾选
```

### 配置2：不同主键位置（启用数值转换）
```
文件1主键列索引: 0 (第1列是ID)
文件2主键列索引: 1 (第2列是ID)
文件1特征起始列: 1
文件2特征起始列: 2
转换特征值为数值: 勾选
```

### 配置3：严格字符串对比（不转换）
```
文件1主键列索引: 0
文件2主键列索引: 0
文件1特征起始列: 1
文件2特征起始列: 1
转换特征值为数值: 不勾选
```

---

## ⚠️ 注意事项

1. 文件大小限制：1GB
2. 支持格式：CSV、XLSX
3. 主键列必须唯一
4. 列索引从0开始
5. 默认启用"转换特征值为数值"

---

## 💡 转换特征值说明

**勾选"转换特征值为数值"时**：
- "8" = 8 = 8.0（都被认为相同）
- 避免因数据类型不同导致的误判

**不勾选时**：
- "8" ≠ 8（被认为不同）
- 严格按字符串对比
