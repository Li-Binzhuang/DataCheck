## 常见问题

### Q: 启动时提示 "No module named 'flask'"

**A**: 需要先激活虚拟环境并安装依赖：
```bash
cd MyDataCheck
source .venv/bin/activate
pip install -r requirements.txt
```

### Q: 如何确认优化是否生效？

**A**: 运行验证脚本：
```bash
python3 MyDataCheck/verify_optimization.py
```

### Q: 处理大文件时还是卡顿怎么办？

**A**: 
1. 检查文件大小（建议 < 50万行）
2. 检查系统内存（建议 > 2GB）
3. 查看日志文件排查问题
