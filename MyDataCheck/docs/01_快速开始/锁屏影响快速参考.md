# 锁屏和关闭页面影响 - 快速参考

## 不会中断 ✅

| 操作 | 影响 |
|------|------|
| 锁屏 | ✅ 不影响，任务继续 |
| 关闭浏览器标签 | ✅ 不影响，任务继续 |
| 关闭浏览器 | ✅ 不影响，任务继续 |
| 客户端休眠 | ✅ 不影响（服务器在另一台机器） |

## 会中断 ❌

| 操作 | 影响 |
|------|------|
| 关闭Web服务器 | ❌ 中断所有任务 |
| Ctrl+C停止服务 | ❌ 中断所有任务 |
| 服务器休眠 | ❌ 中断所有任务 |
| 服务器关机 | ❌ 中断所有任务 |

## 原因

- 任务在 **服务器端** 执行
- 前端只负责显示输出
- 关闭前端不影响后端

## 验证方法

```bash
# 1. 启动任务
# 2. 关闭浏览器
# 3. 查看输出目录
ls -lh MyDataCheck/outputdata/api_comparison/

# 4. 查看进程
ps aux | grep web_app.py
```

## 建议

1. ✅ 可以放心关闭浏览器
2. ✅ 任务执行期间保持Web服务器运行
3. ✅ 使用 `screen` 或 `tmux` 保持会话
4. ✅ 定期检查输出目录

## 后台运行

```bash
# 使用 screen
screen -S mydatacheck
python web_app.py
# Ctrl+A, D 分离

# 使用 nohup
nohup python web_app.py > web.log 2>&1 &
```
