# 全量输出功能 - 快速参考

## 开关
```python
WRITE_FULL_CSV = True  # True=输出全量文件，False=不输出
```

---

## 输出文件

### 分批文件（始终输出）
- `cdc1_features_batch001_1-500.csv`
- `cdc1_features_batch002_501-1000.csv`
- ...

### 全量文件（由开关控制）
- `cdc1_features_all.csv`（第一板块）
- `cdc2_features_all.csv`（第二板块）
- `cdc3_features_all.csv`（第三板块）
- `cdcboss_features_all.csv`（BOSS板块）

---

## 使用

### 启用全量输出（默认）
```python
WRITE_FULL_CSV = True
```
输出：分批文件 + 全量文件

### 禁用全量输出
```python
WRITE_FULL_CSV = False
```
输出：只有分批文件

---

## 配置位置

| 板块 | 行号 | 变量 |
|------|------|------|
| 第一板块 | ~2084 | `WRITE_FULL_CSV = True` |
| 第二板块 | ~1685 | `WRITE_FULL_CSV = True` |
| 第三板块 | ~1004 | `WRITE_FULL_CSV = True` |
| BOSS板块 | ~2317 | `WRITE_FULL_CSV = True` |

## 验证一致性
```python
import pandas as pd
from pathlib import Path

# 读取全量文件
full_df = pd.read_csv("outputs/cdc1_features_all.csv")

# 读取并合并分批文件
batch_files = sorted(Path("outputs").glob("cdc1_features_batch*.csv"))
merged_df = pd.concat([pd.read_csv(f) for f in batch_files], ignore_index=True)

# 比较
print("一致性:", full_df.equals(merged_df))  # 应该是 True
```
