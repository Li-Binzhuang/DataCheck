# CSV合并功能快速指南

## 功能位置
左侧菜单 → 小工具集合 → 合并表格文件

## 两种合并方式

### 📊 纵向合并（追加行）
将多个文件的数据行拼接在一起

**适用场景**：
- 合并多天/多批次的相同结构数据
- 合并分片数据

**要求**：
- 所有文件列结构相同

**示例**：
```
文件1: 100行 × 10列
文件2: 200行 × 10列
文件3: 150行 × 10列
结果:  450行 × 10列
```

---

### 📑 横向合并（追加列）
将多个文件的列拼接在一起

**适用场景**：
- 合并不同来源的特征数据
- 将基础信息和特征表合并

**重要配置**：
- ✅ **推荐勾选"自动移除重复列"**
- 避免主键列重复出现

**示例**：
```
文件1: user_info.csv
  - user_id, name, age (1000行)

文件2: user_features.csv
  - user_id, feature1, feature2, feature3 (1000行)

配置：勾选"自动移除重复列"
结果：user_id, name, age, feature1, feature2, feature3 (1000行)
说明：文件2的user_id被自动移除
```

## 重复列处理

### 方式1：自动检测（推荐）
- 勾选"自动移除重复列"
- 指定列名留空
- 系统自动检测并移除所有重复列

### 方式2：指定列名
- 勾选"自动移除重复列"
- 填写要移除的列名：`user_id,order_id`
- 只移除指定的列，其他重复列保留

### 处理规则
- 始终保留第1个文件的列
- 从第2、3、4...个文件中移除重复列

## 常见问题

### Q1: 横向合并时，文件行数不同怎么办？
A: 直接拼接会导致数据错位。建议：
- 确保所有文件行数相同
- 或使用其他工具基于主键进行关联合并

### Q2: 不勾选"移除重复列"会怎样？
A: pandas会自动给重复列添加后缀：
```
user_id, name, user_id_x, feature1, user_id_y, feature2
```
不推荐这种方式。

### Q3: 纵向合并时列名不完全相同？
A: 会包含所有列，缺失值填充为NaN：
```
文件1: A, B, C
文件2: A, B, D
结果:  A, B, C, D (文件1的D列为NaN，文件2的C列为NaN)
```

### Q4: 可以合并多少个文件？
A: 理论上无限制，但：
- 建议不超过10个文件
- 注意总文件大小不超过5GB

## 操作步骤

1. 选择多个CSV文件（Ctrl/Cmd多选）
2. 选择合并方式
3. 如果是横向合并：
   - ✅ 勾选"自动移除重复列"
   - 可选填写指定列名
4. 输入输出文件名
5. 点击"执行合并"
6. 自动下载结果

## 输出说明

- 文件名格式：`{前缀}_{时间戳}.csv`
- 保存位置：`outputdata/merge_csv/`
- 编码格式：UTF-8-BOM（Excel友好）
- 自动显示：
  - 总行数
  - 总列数
  - 已移除的重复列（如有）
