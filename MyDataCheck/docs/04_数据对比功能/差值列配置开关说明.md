# 差值列配置开关说明

## 功能概述

新增"计算差值"配置开关，允许用户控制是否在对比报告中输出差值列（CSV值 - API值）。

## 配置位置

在接口数据对比页面的每个场景配置中，位于"请求接口时时间加1秒"选项下方：

```
☐ 请求接口时时间加1秒
☐ 计算差值（CSV值 - API值）  ← 新增配置项
```

## 默认值

- **默认状态**: 未勾选（false）
- **原因**: 差值计算是新增功能，为保持向后兼容，默认不启用

## 功能说明

### 勾选时（calculate_difference = true）

分析报告（`_analysis_report.csv`）将包含"差值"列：

| 特征名 | applyId | request_time | CSV值 | API值 | 差值 |
|--------|---------|--------------|-------|-------|------|
| feature1 | 123 | 2025-01-01T12:00:00.000 | 100.5 | 100.0 | 0.5 |
| feature2 | 123 | 2025-01-01T12:00:00.000 | 50.25 | 50.3 | -0.05 |

### 未勾选时（calculate_difference = false）

分析报告不包含"差值"列：

| 特征名 | applyId | request_time | CSV值 | API值 |
|--------|---------|--------------|-------|-------|
| feature1 | 123 | 2025-01-01T12:00:00.000 | 100.5 | 100.0 |
| feature2 | 123 | 2025-01-01T12:00:00.000 | 50.25 | 50.3 |

## 差值计算规则

1. **计算公式**: `差值 = CSV值 - API值`
2. **精度**: 保留6位小数
3. **异常处理**: 
   - 如果CSV值或API值无法转换为数字，差值列显示为空
   - 空值按0处理

## 适用范围

此配置对以下两种对比模式均生效：

1. **完整输出模式**（输出中间文件）
2. **流式对比模式**（仅输出对比报告）

## 配置示例

### JSON配置格式

```json
{
  "scenarios": [
    {
      "name": "场景1",
      "enabled": true,
      "input_csv_file": "test.csv",
      "api_url": "http://example.com/api",
      "calculate_difference": true,  ← 启用差值计算
      "add_one_second": false,
      ...
    }
  ]
}
```

## 使用建议

### 建议启用的场景

- 需要精确分析数值差异的场景
- 需要量化CSV值与API值偏差的场景
- 需要统计差值分布的场景

### 建议不启用的场景

- 只关心是否匹配，不关心具体差值
- 特征值包含大量非数值类型
- 需要减少输出文件大小

## 注意事项

1. 差值列只在分析报告（`_analysis_report.csv`）中输出
2. 特征统计文件（`_feature_stats.csv`）不受此配置影响
3. 此配置不影响对比逻辑，只影响输出格式
4. 修改配置后需要重新执行对比才能生效
