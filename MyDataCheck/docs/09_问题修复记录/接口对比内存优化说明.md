## 技术实现

### 1. 轻量级索引

**原版问题**:
```python
# 存储完整行数据，占用大量内存
api_data_index[key] = (row_index, complete_row)
```

**优化方案**:
```python
# 只存储必要的特征值
api_data_index[key] = {
    "features": {feature_name: value},  # 只存储特征
    "time_now": time_now_value
}
```

**效果**: 索引内存占用降低 60-70%

### 2. 流式写入

**原版问题**:
```python
# 先收集所有异常记录
anomaly_records = []
for row in rows:
    if has_diff:
        anomaly_records.append(record)  # 累积在内存中

# 最后一次性写入
write_all(anomaly_records)
```

**优化方案**:
```python
# 边对比边写入
with CSVStreamWriter(output_path, headers) as writer:
    for row in rows:
        if has_diff:
            writer.write_row(record)  # 立即写入，不累积
```

**效果**: 写入过程内存占用降低 80%

### 3. 及时释放

**优化策略**:
- 处理完一批数据后立即释放
- 不保留不需要的中间结果
- 使用生成器而非列表
