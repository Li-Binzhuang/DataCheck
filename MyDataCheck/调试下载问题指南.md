# 调试下载重复提示问题指南

## 🔍 问题诊断步骤

### 步骤1：打开浏览器控制台

1. 按 `F12` 或 `Ctrl+Shift+I` 打开开发者工具
2. 切换到 `Console` 标签页
3. 清空控制台（点击 🚫 图标）

### 步骤2：执行任务并观察日志

执行一个对比任务，观察控制台输出：

**应该看到的日志：**

```
[AutoDownload] 开始获取 online_comparison 模块的输出文件...
[AutoDownload] taskId: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
[AutoDownload] 找到 X 个未下载的文件，开始下载...
[AutoDownload] 任务已标记为已下载: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

**关键检查点：**

1. ✅ `taskId` 是否有值？（不应该是 `null` 或 `undefined`）
2. ✅ 是否看到 "任务已标记为已下载" 的日志？
3. ✅ 是否有错误信息？

### 步骤3：检查 localStorage

在控制台执行以下命令：

```javascript
// 查看任务下载记录
console.log('任务下载记录:');
console.table(JSON.parse(localStorage.getItem('myDataCheck_downloadedTasks')));

// 查看文件下载记录
console.log('文件下载记录:');
console.table(JSON.parse(localStorage.getItem('myDataCheck_downloadHistory')));
```

**应该看到：**
- 任务ID 和下载时间
- 文件名和下载时间

### 步骤4：刷新页面并观察

刷新页面后，观察控制台输出：

```javascript
[TaskNotification] 检查用户 xxx 的已完成任务...
[TaskNotification] 找到 X 个已完成任务
[TaskNotification] 已下载任务记录: {...}
[TaskNotification] 任务 xxxxxxxx: 已下载
[TaskNotification] 未下载任务数: 0
```

**关键检查点：**

1. ✅ 是否显示 "任务 xxx: 已下载"？
2. ✅ "未下载任务数" 是否为 0？
3. ❌ 如果还是弹出提醒，说明任务没有被正确标记

## 🐛 常见问题排查

### 问题1：taskId 为 null

**症状：**
```
[AutoDownload] taskId: null
[AutoDownload] 未提供 taskId，无法标记任务为已下载
```

**原因：** taskId 没有正确传递

**解决方法：**

检查是哪个模块：

**线上灰度对比模块：**
```javascript
// 在控制台执行
console.log('当前 taskId:', window._currentOnlineTaskId);
```

如果是 `undefined`，说明 taskId 没有被保存。

**API对比模块：**
```javascript
// 在控制台执行
console.log('当前 taskId:', getCurrentTaskId());
```

### 问题2：markTaskDownloaded 函数不存在

**症状：**
```
[AutoDownload] markTaskDownloaded 函数不存在
```

**原因：** JS 文件加载顺序问题

**解决方法：**

检查函数是否存在：
```javascript
console.log('markTaskDownloaded 函数:', typeof markTaskDownloaded);
```

应该输出 `function`，如果是 `undefined`，说明 `task-notification.js` 没有正确加载。

### 问题3：任务记录没有保存

**症状：**
```
[AutoDownload] 任务已标记为已下载: xxx
```
但刷新后 localStorage 中没有记录。

**原因：** localStorage 被禁用或清空

**解决方法：**

检查 localStorage 是否可用：
```javascript
try {
    localStorage.setItem('test', '1');
    localStorage.removeItem('test');
    console.log('localStorage 可用');
} catch (e) {
    console.error('localStorage 不可用:', e);
}
```

### 问题4：任务ID不匹配

**症状：** localStorage 中有记录，但还是弹出提醒

**原因：** 保存的 taskId 和查询的 taskId 不一致

**解决方法：**

对比两个 taskId：
```javascript
// 查看保存的 taskId
const saved = JSON.parse(localStorage.getItem('myDataCheck_downloadedTasks'));
console.log('保存的 taskId:', Object.keys(saved));

// 查看当前任务的 taskId（需要在任务列表中查看）
// 在弹窗中右键检查元素，查看 data-task-id 属性
```

## 🔧 手动修复方法

### 方法1：手动标记任务

如果自动标记失败，可以手动标记：

```javascript
// 1. 查看未下载的任务
fetch('/api/tasks?user_id=YOUR_USER_ID&status=completed')
    .then(r => r.json())
    .then(d => console.table(d.tasks));

// 2. 手动标记任务（替换 TASK_ID）
markTaskDownloaded('TASK_ID');

// 3. 验证
console.log(isTaskDownloaded('TASK_ID'));
```

### 方法2：批量标记所有任务

```javascript
// 获取所有已完成任务并标记
fetch('/api/tasks?user_id=YOUR_USER_ID&status=completed')
    .then(r => r.json())
    .then(d => {
        d.tasks.forEach(task => {
            markTaskDownloaded(task.task_id);
            console.log('已标记:', task.task_id);
        });
    });
```

### 方法3：清空记录重新开始

```javascript
// 清空所有下载记录
localStorage.removeItem('myDataCheck_downloadedTasks');
localStorage.removeItem('myDataCheck_downloadHistory');
console.log('已清空所有下载记录');
```

## 📋 完整诊断脚本

复制以下脚本到控制台，一键诊断：

```javascript
(function() {
    console.log('=== 下载问题诊断 ===\n');
    
    // 1. 检查函数是否存在
    console.log('1. 函数检查:');
    console.log('  - markTaskDownloaded:', typeof markTaskDownloaded);
    console.log('  - isTaskDownloaded:', typeof isTaskDownloaded);
    console.log('  - autoDownloadOutputFiles:', typeof autoDownloadOutputFiles);
    console.log('  - getCurrentTaskId:', typeof getCurrentTaskId);
    console.log('');
    
    // 2. 检查 localStorage
    console.log('2. localStorage 检查:');
    try {
        localStorage.setItem('test', '1');
        localStorage.removeItem('test');
        console.log('  ✅ localStorage 可用');
    } catch (e) {
        console.log('  ❌ localStorage 不可用:', e.message);
    }
    console.log('');
    
    // 3. 查看任务下载记录
    console.log('3. 任务下载记录:');
    try {
        const tasks = JSON.parse(localStorage.getItem('myDataCheck_downloadedTasks'));
        if (tasks && Object.keys(tasks).length > 0) {
            console.table(tasks);
        } else {
            console.log('  (无记录)');
        }
    } catch (e) {
        console.log('  ❌ 读取失败:', e.message);
    }
    console.log('');
    
    // 4. 查看文件下载记录
    console.log('4. 文件下载记录:');
    try {
        const files = JSON.parse(localStorage.getItem('myDataCheck_downloadHistory'));
        if (files && Object.keys(files).length > 0) {
            console.log('  文件数:', Object.keys(files).length);
            console.log('  最近5个:');
            const recent = Object.entries(files).slice(-5);
            recent.forEach(([key, value]) => {
                console.log(`    - ${key}: ${value.downloadTime}`);
            });
        } else {
            console.log('  (无记录)');
        }
    } catch (e) {
        console.log('  ❌ 读取失败:', e.message);
    }
    console.log('');
    
    // 5. 检查当前 taskId
    console.log('5. 当前 taskId:');
    if (typeof getCurrentTaskId === 'function') {
        console.log('  - API模块:', getCurrentTaskId());
    }
    if (typeof window._currentOnlineTaskId !== 'undefined') {
        console.log('  - Online模块:', window._currentOnlineTaskId);
    }
    console.log('');
    
    console.log('=== 诊断完成 ===');
})();
```
