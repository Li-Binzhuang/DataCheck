# 全量输出文件名修改说明

## 修改内容

将全量输出文件名从 `_full_data.csv` 改为 `_all.csv`

---

## 修改前后对比

### 修改前
- `cdc1_features_full_data.csv`
- `cdc2_features_full_data.csv`
- `cdc3_features_full_data.csv`
- `cdcboss_features_full_data.csv`

### 修改后 ✅
- `cdc1_features_all.csv`
- `cdc2_features_all.csv`
- `cdc3_features_all.csv`
- `cdcboss_features_all.csv`

---

## 文件命名规则

### 分批文件
格式：`{板块前缀}_features_batch{批次号:03d}_{起始行}-{结束行}.csv`

示例：
- `cdc1_features_batch001_1-500.csv`
- `cdc1_features_batch002_501-1000.csv`
- `cdc1_features_batch026_12501-12546.csv`

### 全量文件
格式：`{板块前缀}_features_all.csv`

示例：
- `cdc1_features_all.csv`（第一板块）
- `cdc2_features_all.csv`（第二板块）
- `cdc3_features_all.csv`（第三板块）
- `cdcboss_features_all.csv`（BOSS板块）

## 使用方法

### 1. 在Jupyter中运行板块脚本

运行任意板块的衍生脚本，会自动生成：
- 分批文件（每个500条）
- 全量文件（所有数据）

### 2. 查看输出

```bash
ls -lh CDC/outputs/*_all.csv
```

应该看到：
```
cdc1_features_all.csv
cdc2_features_all.csv
cdc3_features_all.csv
cdcboss_features_all.csv
```

## 验证方法

### 检查文件是否存在
```python
from pathlib import Path

output_dir = Path("CDC/outputs")

# 检查全量文件
all_files = [
    "cdc1_features_all.csv",
    "cdc2_features_all.csv",
    "cdc3_features_all.csv",
    "cdcboss_features_all.csv"
]

for filename in all_files:
    filepath = output_dir / filename
    if filepath.exists():
        size_mb = filepath.stat().st_size / (1024 * 1024)
        print(f"✅ {filename}: {size_mb:.2f} MB")
    else:
        print(f"❌ {filename}: 不存在")
```

### 验证数据一致性
```python
import pandas as pd
from pathlib import Path

# 读取全量文件
full_df = pd.read_csv("CDC/outputs/cdc1_features_all.csv")

# 读取并合并分批文件
batch_files = sorted(Path("CDC/outputs").glob("cdc1_features_batch*.csv"))
merged_df = pd.concat([pd.read_csv(f) for f in batch_files], ignore_index=True)

# 比较
print(f"全量文件行数: {len(full_df)}")
print(f"分批文件合并后行数: {len(merged_df)}")
print(f"数据一致性: {full_df.equals(merged_df)}")  # 应该是 True
```

## 注意事项

### 1. 文件覆盖
每次运行脚本都会覆盖之前的全量文件，注意备份重要数据。

### 2. 磁盘空间
全量文件可能较大（几十MB到几百MB），确保有足够的磁盘空间。

### 3. 内存使用
生成全量文件时，所有数据会在内存中处理，确保有足够的内存。

### 4. 开关控制
如果不需要全量文件，可以设置：
```python
WRITE_FULL_CSV = False
```
