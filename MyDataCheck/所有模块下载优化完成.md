## 🔧 核心修改

### 问题根源
所有模块都存在相同的问题：`taskId` 在被使用之前就被清除或丢失。

### 解决方案
在每个模块中使用局部变量保存 `taskId`，避免被清除：

```javascript
// 1. 声明局部变量
let savedTaskId = null;

// 2. 在收到 start 消息时保存
if (data.type === 'start' && data.task_id) {
    savedTaskId = data.task_id;
}

// 3. 在下载时使用保存的 taskId
setTimeout(() => autoDownloadOutputFiles('module_name', 2, savedTaskId), 1000);
```
