## 修复后的代码

```python
    _features_to_write = features.rename(columns=_rename_map_for_write).reset_index()

    _features_to_write[_round_cols] = _features_to_write[_round_cols].apply(pd.to_numeric, errors="coerce").round(6).fillna(-999.0)  # ✅ 正确缩进
```
