## 故障排查

### 如果优化后仍然很慢

**可能原因1：未正确切换版本**
```bash
# 检查是否使用了优化版本
grep "data_comparator_optimized" web/routes/compare_routes.py

# 如果没有，手动修改并重启
```

**可能原因2：文件过大**
```bash
# 检查文件大小
ls -lh inputdata/data_comparison/

# 如果超过5GB，建议分块处理
```

**可能原因3：特征列过多**
```bash
# 检查列数
head -1 inputdata/data_comparison/file1.csv | tr ',' '\n' | wc -l

# 如果超过5000列，建议筛选必要列
```

### 如果出现错误

**错误1：ModuleNotFoundError**
```
解决：确保 data_comparator_optimized.py 存在
ls -la data_comparison/job/data_comparator_optimized.py
```

**错误2：内存不足**
```
解决：增加系统内存或使用流式处理
```

**错误3：编码错误**
```
解决：检查文件编码，确保为UTF-8
file -I inputdata/data_comparison/*.csv
```
