# 数据对比输入目录

## 说明
此目录用于存放待对比的CSV或XLSX文件。

## 使用方法
1. 将Sql文件和接口文件放在此目录下
2. 在配置文件中指定文件名
3. 执行数据对比

## 文件要求
- 支持格式：CSV、XLSX
- 编码：UTF-8、GBK、GB2312
- 确保文件包含主键列和特征列
- 主键列的值应该唯一

## 示例文件结构
```
inputdata/data_comparison/
├── sql_data_20260126.csv      # Sql文件
├── api_data_20260126.csv      # 接口文件
└── README.md                  # 本文档
```

## 注意事项
- 文件名不要包含特殊字符
- 确保文件编码正确
- 大文件可能需要较长处理时间
