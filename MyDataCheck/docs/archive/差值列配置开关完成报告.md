## 功能特性

### 1. 配置位置
- 在接口数据对比页面的每个场景配置中
- 位于"请求接口时时间加1秒"选项下方
- 复选框标签：`☐ 计算差值（CSV值 - API值）`

### 2. 默认行为
- **默认状态**: 未勾选（`false`）
- **原因**: 保持向后兼容，不影响现有配置

### 3. 输出差异

#### 勾选时（`calculate_difference = true`）
分析报告包含"差值"列：
```
特征名, applyId, request_time, CSV值, API值, 差值
```

#### 未勾选时（`calculate_difference = false`）
分析报告不包含"差值"列：
```
特征名, applyId, request_time, CSV值, API值
```

### 4. 差值计算规则
- **公式**: `差值 = CSV值 - API值`
- **精度**: 保留6位小数
- **异常处理**: 无法转换为数字时，差值列显示为空

### 5. 适用范围
- ✅ 完整输出模式（输出中间文件）
- ✅ 流式对比模式（仅输出对比报告）

## 配置示例

### JSON配置
```json
{
  "scenarios": [
    {
      "name": "场景1",
      "enabled": true,
      "calculate_difference": true,
      ...
    }
  ]
}
```


## 使用建议

### 建议启用
- 需要精确分析数值差异
- 需要量化偏差程度
- 需要统计差值分布

### 建议不启用
- 只关心是否匹配
- 特征值包含大量非数值类型
- 需要减少输出文件大小

## 注意事项

1. 差值列只在分析报告（`_analysis_report.csv`）中输出
2. 特征统计文件（`_feature_stats.csv`）不受影响
3. 此配置不影响对比逻辑，只影响输出格式
4. 修改配置后需要重新执行对比才能生效
