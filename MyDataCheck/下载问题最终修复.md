# 下载重复提示问题 - 最终修复

## 🐛 根本原因

**问题：** `taskId` 在被使用之前就被清除了

**详细分析：**

1. 任务完成时，后端发送 `type: 'end'` 消息
2. `processLine` 函数收到 `end` 消息，调用 `clearCurrentTaskId()`
3. `clearCurrentTaskId()` 将全局变量 `currentTaskId` 设置为 `null`
4. SSE 流结束，`readStream` 的 `done` 分支被触发
5. 调用 `getCurrentTaskId()` 获取 taskId，但此时已经是 `null`
6. `autoDownloadOutputFiles` 收到 `taskId = null`
7. 无法标记任务为已下载
8. 刷新页面后，任务仍然显示为未下载，弹出提醒

**时序图：**

```
1. 收到 end 消息
   ↓
2. processLine() 调用 clearCurrentTaskId()
   ↓
3. currentTaskId = null
   ↓
4. SSE 流结束 (done = true)
   ↓
5. readStream() 调用 getCurrentTaskId()
   ↓
6. 返回 null ❌
   ↓
7. autoDownloadOutputFiles(module, 2, null)
   ↓
8. 无法标记任务 ❌
```

## ✅ 解决方案

**核心思路：** 在 `clearCurrentTaskId()` 被调用之前保存 `taskId`

**实现方法：** 使用局部变量 `savedTaskId` 保存 taskId

### 修改的文件

**static/js/api-compare.js**

**修改前：**
```javascript
const reader = response.body.getReader();
const decoder = new TextDecoder();
let buffer = '';

function readStream() {
    reader.read().then(({ done, value }) => {
        if (done) {
            // ...
            const taskId = getCurrentTaskId(); // ❌ 此时已经是 null
            setTimeout(() => autoDownloadOutputFiles('api_comparison', 2, taskId), 1000);
            return;
        }
        // ...
    });
}

function processLine(line, tab) {
    if (data.type === 'start') {
        if (data.task_id && typeof setCurrentTaskId === 'function') {
            setCurrentTaskId(data.task_id);
        }
    } else if (data.type === 'end') {
        if (typeof clearCurrentTaskId === 'function') {
            clearCurrentTaskId(); // ❌ 清除了 taskId
        }
    }
}
```

**修改后：**
```javascript
const reader = response.body.getReader();
const decoder = new TextDecoder();
let buffer = '';
let savedTaskId = null; // ✅ 新增：保存 taskId

function readStream() {
    reader.read().then(({ done, value }) => {
        if (done) {
            // ...
            // ✅ 使用保存的 taskId
            setTimeout(() => autoDownloadOutputFiles('api_comparison', 2, savedTaskId), 1000);
            return;
        }
        // ...
    });
}

function processLine(line, tab) {
    if (data.type === 'start') {
        if (data.task_id) {
            savedTaskId = data.task_id; // ✅ 保存到局部变量
            if (typeof setCurrentTaskId === 'function') {
                setCurrentTaskId(data.task_id);
            }
        }
    } else if (data.type === 'end') {
        if (typeof clearCurrentTaskId === 'function') {
            clearCurrentTaskId(); // 清除全局变量，但局部变量仍然保留
        }
    }
}
```
