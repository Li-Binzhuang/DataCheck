# 前200条记录输出功能 - 使用说明

## 🎯 功能说明

CDC项目的四个板块Notebook现在会额外输出一个包含**前200条记录**的小文件，方便快速查看和测试。

## 📁 输出文件

运行Notebook后，在 `outputs/` 目录会生成：

| 板块 | 样本文件 |
|-----|---------|
| 第一板块 | `cdc1_features_consultas_sample200.csv` |
| 第二板块 | `cdc2_features_creditos_sample200.csv` |
| 第三板块 | `cdc3_features_clave_prevencion_sample200.csv` |
| BOSS板块 | `cdcboss_features_sample200.csv` |

## 🚀 快速开始

1. 打开任意Notebook
2. 运行所有代码格
3. 查看 `outputs/` 目录

就这么简单！

## ⚙️ 配置选项

在Notebook的第4个代码格中：

```python
WRITE_SAMPLE_200 = True   # 输出前200条样本（默认）
WRITE_SAMPLE_200 = False  # 不输出样本
```

## 💡 为什么需要这个功能？

- ✅ **快速**: 200条记录的文件秒开，全量文件需要等待
- ✅ **轻量**: 文件大小只有全量的1.6%
- ✅ **便捷**: 开发调试时不需要加载12546条记录
- ✅ **一致**: 数据格式与全量文件完全相同

## 📊 文件大小对比

```
全量特征表: ~50MB  (12546条)
前200条样本: ~0.8MB (200条)
```

样本文件只有全量文件的 **1.6%**！

## ❓ 常见问题

**Q: 样本文件包含哪些记录？**  
A: 前200个apply_id的特征记录。

**Q: 会影响全量文件吗？**  
A: 不会，两者独立输出。

**Q: 如何关闭样本输出？**  
A: 将 `WRITE_SAMPLE_200` 改为 `False`。
