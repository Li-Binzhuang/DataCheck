## 整理脚本

### organize_files.sh
原始的文件整理脚本，可以整理特定模式的文件：
- 迁移相关文档 → `docs/migration/`
- UI相关文档 → `docs/ui/`
- 清理相关文档 → `docs/cleanup/`
- 测试文件 → `tests/`
- 工具脚本 → `scripts/`
- 备份文件 → `backups/`

**使用方法**：
```bash
cd MyDataCheck
./organize_files.sh
```

### organize_all_files.sh
增强版的文件整理脚本，功能更全面：
- 自动检测所有需要整理的文件
- 提供详细的整理统计
- 检查异常文件
- 显示整理后的目录结构

**使用方法**：
```bash
cd MyDataCheck
./organize_all_files.sh
```

## 维护建议

### 定期整理
建议每月执行一次文件整理：
```bash
cd MyDataCheck
./organize_all_files.sh
```

### 文件管理规范
1. **备份文件**：统一放到 `backups/` 目录
2. **测试文件**：统一放到 `tests/` 目录
3. **工具脚本**：统一放到 `scripts/` 目录
4. **临时文档**：及时归档或删除
5. **新文档**：根据类型放到对应的 `docs/` 子目录

### 避免根目录混乱
- ✅ 核心文件保持在根目录
- ✅ 其他文件分类存放
- ✅ 定期清理临时文件
- ✅ 使用整理脚本自动化
