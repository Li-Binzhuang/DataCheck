# 数据对比界面标签更新说明

## 更新时间
2026-01-27

## 更新内容

### 文件上传标签

**更新前**：
- Sql文件 (CSV/XLSX)
- 接口文件 (CSV/XLSX)

**更新后**：
- sql data (CSV/XLSX)
- 从库/灰度/线上文件 (CSV/XLSX)

### 主键列索引标签

**更新前**：
- Sql文件主键列索引 (A列=0)
- 接口文件主键列索引 (A列=0)

**更新后**：
- sql data主键列索引 (A列=0)
- 从库/灰度/线上文件主键列索引 (A列=0)

### 特征起始列索引标签

**更新前**：
- Sql文件特征起始列索引
- 接口文件特征起始列索引

**更新后**：
- sql data特征起始列索引
- 从库/灰度/线上文件特征起始列索引

## 更新原因

1. **更准确的命名**：
   - "sql data"更准确地描述了数据来源（SQL查询结果）
   - "从库/灰度/线上文件"更清楚地说明了对比目标的多种可能来源

2. **更好的用户理解**：
   - 避免"Sql文件"和"接口文件"的混淆
   - 明确说明第二个文件可以是从库、灰度环境或线上环境的数据

3. **统一术语**：
   - 与实际使用场景保持一致
   - 便于用户理解数据对比的目的

## 使用示例

### 场景1：对比SQL查询结果与从库数据
- **sql data**: 上传SQL查询导出的CSV文件
- **从库/灰度/线上文件**: 上传从库导出的CSV文件

### 场景2：对比SQL查询结果与灰度环境数据
- **sql data**: 上传SQL查询导出的CSV文件
- **从库/灰度/线上文件**: 上传灰度环境导出的CSV文件

### 场景3：对比SQL查询结果与线上环境数据
- **sql data**: 上传SQL查询导出的CSV文件
- **从库/灰度/线上文件**: 上传线上环境导出的CSV文件

## 向后兼容

所有后端API和配置文件格式保持不变，只是界面显示的标签进行了更新，不影响现有功能和数据。
