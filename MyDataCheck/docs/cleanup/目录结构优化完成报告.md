## 软链接说明

为了便于使用，在根目录创建了常用脚本的软链接：

```bash
start_web.sh -> scripts/startup/start_web.sh
stop_web.sh -> scripts/startup/stop_web.sh
```

**优势**:
- 保持根目录整洁
- 便于快速访问
- 实际文件统一管理

**使用方法**:
```bash
# 在根目录直接使用（推荐）
./start_web.sh
./stop_web.sh

# 或使用完整路径
./scripts/startup/start_web.sh
./scripts/startup/stop_web.sh
```
