## 📊 项目简介

MyDataCheck 是一个专业的数据对比工具平台，提供美观的Web界面和强大的数据处理能力。

### 核心功能
- 📡 **接口数据对比**：对比API返回数据与预期数据
- 🌐 **线上灰度落数对比**：对比线上环境与灰度环境数据
- 📊 **数据对比**：对比CSV/XLSX文件的数据差异
- 📦 **PKL文件解析**：解析和转换PKL文件

---

## 🚀 快速开始

### 启动服务
```bash
cd MyDataCheck
./start_web.sh
```

### 访问界面
```
http://localhost:5001
```

**注意**：由于macOS的AirPlay Receiver占用5000端口，Web服务使用5001端口。详见 [端口配置说明.md](端口配置说明.md)

### 停止服务
```bash
./stop_web.sh
```

## 💡 使用技巧

### 接口对比
1. 添加场景（支持多个）
2. 上传CSV文件（包含测试数据）
3. 配置API地址和参数
4. 设置线程数和超时时间
5. 点击"执行对比"
6. 查看对比结果

### 数据对比
1. 上传两个文件（CSV/XLSX）
2. 设置主键列索引（A列=0）
3. 设置特征起始列索引
4. 勾选"转换特征值为数值"（推荐）
5. 点击"执行对比"
6. 下载对比报告

### PKL解析
1. 上传PKL文件
2. 点击"解析文件"查看内容
3. 选择转换格式：
   - 标准CSV
   - CDC V2核心CSV
4. 下载转换结果

## 🐛 常见问题

### Q1: 服务启动失败？
```bash
# 检查端口占用
lsof -i :5000

# 杀死占用进程
kill -9 <PID>

# 重新启动
./start_web.sh
```

### Q2: 文件上传失败？
- 检查文件格式（CSV/XLSX/PKL）
- 检查文件编码（UTF-8）
- 检查文件大小（<100MB）

### Q3: 对比结果不准确？
- 勾选"转换特征值为数值"
- 检查主键列索引
- 检查特征起始列索引

### Q4: 界面显示异常？
- 清除浏览器缓存
- 使用Chrome/Safari浏览器
- 检查网络连接

### Q5: 保存配置报错？
- 确保服务已重启（最新版本已修复）
- 检查配置文件权限
- 查看 [变量名问题完整修复说明.md](变量名问题完整修复说明.md)

### Q6: 上传文件报错？
- 确保服务已重启（最新版本已修复）
- 检查文件格式（CSV/XLSX/PKL）
- 检查文件大小（<100MB）
- 查看 [变量名问题完整修复说明.md](变量名问题完整修复说明.md)
