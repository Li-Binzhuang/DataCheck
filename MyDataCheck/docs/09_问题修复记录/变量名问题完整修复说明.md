# 变量名问题完整修复说明

## 问题概述

Web界面多个功能报错，都是因为变量名不一致导致的：
- ❌ 数据对比 - 保存配置报错：`name 'data_comparison_dir' is not defined`
- ❌ 数据对比 - 上传文件报错：`name 'compare_input_dir' is not defined`
- ❌ 接口对比 - 可能报错：`name 'api_input_dir' is not defined`
- ❌ 线上对比 - 可能报错：`name 'online_input_dir' is not defined`

---

## 根本原因

**变量命名不一致**：
- `web/config.py` 中定义的是**大写变量**（Python常量命名规范）
- `web/routes/*.py` 中使用的是**小写变量**（普通变量命名）

### 示例
```python
# web/config.py 中定义
DATA_COMPARISON_DIR = os.path.join(SCRIPT_DIR, "data_comparison")
COMPARE_INPUT_DIR = os.path.join(INPUT_DATA_DIR, "data_comparison")

# web/routes/compare_routes.py 中使用
config_path = os.path.join(data_comparison_dir, "config.json")  # ❌ 未定义
file_path = os.path.join(compare_input_dir, filename)           # ❌ 未定义
```

---

## 修复方案

在每个路由文件中添加小写别名，保持向后兼容。

### 1. compare_routes.py（数据对比）

#### 修复前
```python
from web.config import (
    SCRIPT_DIR, JOB_DIR, ONLINE_JOB_DIR, DATA_COMPARISON_JOB_DIR, COMMON_DIR,
    API_OUTPUT_DIR, ONLINE_OUTPUT_DIR, COMPARE_OUTPUT_DIR,
    API_INPUT_DIR, ONLINE_INPUT_DIR, COMPARE_INPUT_DIR
)

compare_bp = Blueprint('compare_routes', __name__)
```

#### 修复后
```python
from web.config import (
    SCRIPT_DIR, JOB_DIR, ONLINE_JOB_DIR, DATA_COMPARISON_JOB_DIR, COMMON_DIR,
    API_OUTPUT_DIR, ONLINE_OUTPUT_DIR, COMPARE_OUTPUT_DIR,
    API_INPUT_DIR, ONLINE_INPUT_DIR, COMPARE_INPUT_DIR, DATA_COMPARISON_DIR
)

compare_bp = Blueprint('compare_routes', __name__)

# 为了兼容性，创建小写别名
data_comparison_dir = DATA_COMPARISON_DIR
data_comparison_job_dir = DATA_COMPARISON_JOB_DIR
compare_input_dir = COMPARE_INPUT_DIR
compare_output_dir = COMPARE_OUTPUT_DIR
```

## 为什么使用别名而不是直接改名？

### 优点
1. **向后兼容**：不破坏现有代码
2. **快速修复**：只需添加几行代码
3. **易于理解**：清楚地标记了兼容性代码
4. **便于重构**：未来可以统一改为大写

### 示例
```python
# 方案1：使用别名（当前方案）✅
from web.config import DATA_COMPARISON_DIR
data_comparison_dir = DATA_COMPARISON_DIR  # 兼容旧代码

# 方案2：直接使用大写（需要修改所有引用）❌
from web.config import DATA_COMPARISON_DIR
config_path = os.path.join(DATA_COMPARISON_DIR, "config.json")  # 需要改100+处
```

## 长期优化建议

### 1. 统一使用大写变量（推荐）
```python
# 所有路由文件中直接使用大写
from web.config import DATA_COMPARISON_DIR, COMPARE_INPUT_DIR
config_path = os.path.join(DATA_COMPARISON_DIR, "config.json")
```

### 2. 使用配置类（更优雅）
```python
# web/config.py
class Config:
    DATA_COMPARISON_DIR = "..."
    COMPARE_INPUT_DIR = "..."

# web/routes/compare_routes.py
from web.config import Config
config_path = os.path.join(Config.DATA_COMPARISON_DIR, "config.json")
```
