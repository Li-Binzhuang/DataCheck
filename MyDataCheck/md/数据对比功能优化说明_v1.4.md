# 数据对比功能优化说明 v1.4

## 更新时间
2026-01-26

## 优化内容

### 1. 修复转换函数处理引号问题

**问题描述：**
在启用"转换特征值为数值"功能后，发现部分数据中包含多余的英文双引号，导致转换不完整。

**原因分析：**
之前的 `_convert_string_to_number()` 函数虽然能去除字符串两端的引号，但在转换失败时会返回原值（包含引号），导致非数字字符串的引号没有被去除。

**解决方案：**
优化转换函数，确保无论是否能转换为数值，都会去除字符串两端的引号：

```python
def _convert_string_to_number(value: Any) -> Any:
    """
    尝试将字符串转换为数值类型，并去除多余的引号
    
    Args:
        value: 输入值
    
    Returns:
        如果可以转换为数值，返回数值；否则返回去除引号后的字符串
    """
    if value is None or value == "":
        return value
    
    # 如果已经是数值类型，直接返回
    if isinstance(value, (int, float)):
        return value
    
    # 转换为字符串并去除空格
    str_value = str(value).strip()
    
    # 处理空字符串
    if not str_value:
        return value
    
    # 去除字符串两端的双引号（单引号也一并处理）
    # 例如: "8" -> 8, '"8"' -> 8, "'8'" -> 8, "abc" -> abc
    if len(str_value) >= 2:
        # 去除外层的双引号或单引号
        if (str_value[0] == '"' and str_value[-1] == '"') or \
           (str_value[0] == "'" and str_value[-1] == "'"):
            str_value = str_value[1:-1].strip()
    
    # 再次检查是否为空
    if not str_value:
        return value
    
    # 尝试转换为数值
    try:
        # 先尝试转换为浮点数
        float_value = float(str_value)
        # 如果浮点数等于其整数形式，返回整数
        if float_value == int(float_value):
            return int(float_value)
        # 否则返回浮点数
        return float_value
    except (ValueError, TypeError):
        # 无法转换为数值，返回去除引号后的字符串
        return str_value
```

**转换规则：**
- `"8"` → 去除双引号 → `8` → 转换为数字 `8`
- `'"8"'` → 去除外层引号 → `"8"` → 去除双引号 → `8` → 转换为数字 `8`
- `'8'` → 去除单引号 → `8` → 转换为数字 `8`
- `"8.5"` → 去除双引号 → `8.5` → 转换为浮点数 `8.5`
- `"abc"` → 去除双引号 → `abc` → 无法转换，返回字符串 `abc`
- `"  8  "` → 去除双引号 → `  8  ` → 去除空格 → `8` → 转换为数字 `8`

**测试结果：**
所有测试用例通过，包括：
- ✅ 普通字符串数字
- ✅ 带双引号的字符串数字
- ✅ 带单引号的字符串数字
- ✅ 整数和浮点数
- ✅ 字符串浮点数
- ✅ 带双引号的字符串浮点数
- ✅ 空字符串和None值
- ✅ 非数字字符串
- ✅ 带双引号的非数字字符串
- ✅ 带空格的字符串数字
- ✅ 带双引号和空格的字符串数字

## 影响范围

### 修改的文件
- `MyDataCheck/online_comparison/job/data_comparator.py`
  - 优化 `_convert_string_to_number()` 函数

### 功能影响
- 启用"转换特征值为数值"功能后，所有字符串值都会去除两端的引号
- 数值类型的字符串会被转换为对应的数值类型
- 非数值类型的字符串会去除引号后保持为字符串
- 不影响未启用"转换特征值为数值"功能的场景

## 使用说明

### 在Web界面中使用
1. 在"数据对比"或"线上灰度落数对比"页面
2. 勾选"转换特征值为数值"复选框（默认勾选）
3. 上传文件并执行对比
4. 系统会自动去除数据中的引号并尝试转换为数值

### 转换行为
- **数值字符串**：`"8"` → `8`，`"8.5"` → `8.5`
- **非数值字符串**：`"abc"` → `abc`（去除引号但保持字符串类型）
- **已经是数值**：`8` → `8`（不做处理）
- **空值**：`""` → `""`，`None` → `None`（不做处理）

## 注意事项

1. **引号处理**：只处理字符串两端的引号，不处理字符串中间的引号
2. **空格处理**：会自动去除字符串两端的空格
3. **类型转换**：整数形式的浮点数会转换为整数（如 `8.0` → `8`）
4. **兼容性**：不影响未启用转换功能的场景
