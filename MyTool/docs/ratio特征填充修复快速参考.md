# Ratio 特征填充修复 - 快速参考

## 问题

特征值出现 `-333`、`-166.5` 等错误值，应该是 `-999`

## 原因

```python
cnt = -999  # 已填充
ratio = -999 / 3 = -333  # ❌ 错误
```

## 修复

```python
# 修复前（错误）
ratio = cnt.div(total, axis=0).fillna(-999.0)

# 修复后（正确）
ratio = cnt.replace(-999, np.nan).div(total.replace(0, np.nan), axis=0).fillna(-999.0)
```
