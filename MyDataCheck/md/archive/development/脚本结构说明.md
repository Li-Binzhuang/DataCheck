# 场景1_接口数据对比 - 脚本结构说明

## 当前脚本列表

### ✅ 主入口脚本（保留）

**执行对比流程.py**
- 位置：根目录
- 功能：主执行脚本，整合所有功能
- 用途：用户直接运行此脚本执行完整流程
- 依赖：调用job文件夹下的功能模块

---

### ✅ 功能模块脚本（保留，在job文件夹下）

**job/配置管理.py**
- 功能：处理JSON配置文件的加载和单场景配置的构建
- 被调用：执行对比流程.py
- 用途：配置管理相关功能

**job/流程执行器.py**
- 功能：执行接口数据获取和对比的各个步骤
- 被调用：执行对比流程.py
- 用途：核心流程执行逻辑

**job/检测列索引.py**
- 功能：自动检测CSV文件中cust_no和时间字段的列索引
- 被调用：流程执行器.py
- 用途：列索引检测工具

**job/获取接口数据.py**
- 功能：读取CSV文件，请求接口，输出接口全部数据
- 被调用：流程执行器.py
- 用途：接口数据获取功能

**job/对比接口数据.py**
- 功能：对比原始数据和接口返回的数据
- 被调用：流程执行器.py
- 用途：数据对比功能

---

### ❌ 已删除的冗余脚本

**job/接口校验.py** - 已删除

## 使用方式

### 推荐方式（使用主脚本）

```bash
python 执行对比流程.py
```

### 独立运行功能模块（可选）

如果需要单独运行某个功能模块：

```bash
# 检测列索引
python job/检测列索引.py [CSV文件路径] [配置文件路径]

# 获取接口数据
python job/获取接口数据.py

# 对比接口数据
python job/对比接口数据.py
```
