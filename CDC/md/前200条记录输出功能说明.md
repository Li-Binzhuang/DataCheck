# CDC项目：前200条记录输出功能说明

## 功能概述

为CDC项目的四个板块Notebook添加了前200条记录输出功能，方便快速查看和测试特征数据。

## 修改内容

### 1. 第一板块（consultas - 查询板块）
- **Notebook**: `CDC/第一板块衍生.ipynb`
- **新增开关**: `WRITE_SAMPLE_200 = True`
- **输出文件**: `outputs/cdc1_features_consultas_sample200.csv`
- **说明**: 输出前200条查询特征记录

### 2. 第二板块（creditos - 信贷板块）
- **Notebook**: `CDC/第二板块衍生.ipynb`
- **新增开关**: `WRITE_SAMPLE_200 = True`
- **输出文件**: `outputs/cdc2_features_creditos_sample200.csv`
- **说明**: 输出前200条信贷特征记录

### 3. 第三板块（clavePrevencion - 预防类型板块）
- **Notebook**: `CDC/第三板块衍生.ipynb`
- **新增开关**: `WRITE_SAMPLE_200 = True`
- **输出文件**: `outputs/cdc3_features_clave_prevencion_sample200.csv`
- **说明**: 输出前200条预防类型特征记录

### 4. BOSS板块
- **Notebook**: `CDC/BOSS板块衍生.ipynb`
- **新增开关**: `WRITE_SAMPLE_200 = True`
- **输出文件**: `outputs/cdcboss_features_sample200.csv`
- **说明**: 输出前200条BOSS特征记录

## 使用方法

### 方式一：使用默认设置（推荐）
1. 打开任意一个Notebook
2. 直接运行所有代码格
3. 在 `outputs/` 目录下会自动生成三个文件：
   - `*_flat.csv`: 平铺明细表（全量）
   - `*_features_*.csv`: 特征表（全量）
   - `*_features_*_sample200.csv`: 前200条记录（用于快速查看）

### 方式二：自定义输出
如果不想输出前200条记录，可以在Notebook的第4个代码格中修改：
```python
WRITE_SAMPLE_200 = False  # 改为 False 跳过样本输出
```

## 输出文件对比

| 文件类型 | 记录数 | 用途 | 文件大小 |
|---------|--------|------|---------|
| 全量特征表 | 12546条 | 完整的特征数据，用于模型训练 | 较大 |
| 前200条样本 | 200条 | 快速查看特征结构和数据质量 | 很小 |

## 优势

1. **快速查看**: 200条记录的文件打开速度快，方便快速检查特征是否正确
2. **节省时间**: 不需要等待全量文件加载，提高开发效率
3. **便于测试**: 可以用小文件快速测试数据处理流程
4. **灵活控制**: 通过开关可以随时启用或禁用样本输出

## 技术细节

### 实现逻辑
```python
# 1. 添加开关
WRITE_SAMPLE_200 = True

# 2. 在输出全量特征后，添加样本输出
if WRITE_SAMPLE_200:
    _features_sample200 = _features_to_write.head(200)  # 取前200条
    _features_sample200.to_csv(
        features_sample200_path,
        index=False,
        encoding="utf-8-sig",
    )
```

### 数据一致性
- 样本数据与全量数据完全一致（只是行数不同）
- 使用相同的列名、数据类型和格式
- 保留2位小数的规则也适用于样本数据

## 注意事项

1. **记录顺序**: 样本文件包含的是前200条记录，顺序与全量文件一致
2. **apply_id**: 样本文件中的apply_id与全量文件中的前200个apply_id相同
3. **文件编码**: 所有输出文件统一使用 `utf-8-sig` 编码，确保Excel正确显示中文
4. **输出目录**: 所有文件输出到 `outputs/` 目录，该目录会自动创建
