# 科学计数法极小值处理 - 快速参考

## 一句话说明

CSV 文件中的 `0E-8` 等科学计数法极小值会自动视为 `0` 进行对比。

## 会被转换为 0

✅ `0E-8` → `0`  
✅ `0.0E-10` → `0`  
✅ `1E-20` → `0` (极小值)  
✅ `0e-8` → `0` (小写)

## 不会被转换

❌ `0.00000001` → 保持原值 (小数格式)  
❌ `1E-5` → 保持原值 (0.00001，不够小)  
❌ `123.456` → 保持原值 (正常数值)

## 对比示例

| CSV 值 | 接口值 | 结果 | 说明 |
|--------|--------|------|------|
| `0E-8` | `0` | ✅ 一致 | 科学计数法的 0 |
| `1E-20` | `0` | ✅ 一致 | 极小值视为 0 |
| `1E-5` | `0.00001` | ✅ 一致 | 正常科学计数法 |
| `1E-5` | `0` | ❌ 不一致 | 不够小，不视为 0 |
| `0.00000001` | `0` | ❌ 不一致 | 小数格式不转换 |
