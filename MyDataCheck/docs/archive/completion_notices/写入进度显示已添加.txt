🎯 技术特点:

• 同行刷新: 使用 \r 实现，不产生大量输出
• 更新频率: 每100行（或1000行）更新一次
• 性能影响: < 0.1%，几乎可忽略
• 自动换行: 完成后自动换行，保持输出整洁

📖 使用说明:

1. 接口数据对比（自动启用）
   - 无需配置，自动显示进度
   - 写入接口数据文件时自动显示

2. 批量写入器（手动启用）
   
   from common.csv_tool import CSVBatchWriter
   
   # 显示进度（知道总行数）
   with CSVBatchWriter("output.csv", headers, 
                       show_progress=True, 
                       total_rows=len(data)) as writer:
       for row in data:
           writer.write_row(row)
   
   # 显示进度（不知道总行数）
   with CSVBatchWriter("output.csv", headers, 
                       show_progress=True) as writer:
       for row in data:
           writer.write_row(row)

3. 测试验证
   
   cd MyDataCheck
   python test_progress_display.py

💡 使用建议:

✅ 推荐启用:
  • 大文件写入（> 1000行）
  • 长时间运行的任务
  • 用户交互场景

❌ 不推荐启用:
  • 小文件写入（< 100行）
  • 批量自动化任务
  • 需要保持输出简洁的场景
