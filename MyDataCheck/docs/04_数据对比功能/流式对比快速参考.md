# 流式对比快速参考

## 问题
流式对比和完整模式的对比逻辑一样吗？

## 答案
✅ **完全一样，对比逻辑完全一致**

---

## 核心对比逻辑

### 相同的对比函数
```python
def compare_values(self, val1: str, val2: str) -> bool:
    if val1 == val2:
        return True
    
    # 浮点数对比（容忍误差 1e-6）
    try:
        num1 = float(val1)
        num2 = float(val2)
        return abs(num1 - num2) < 1e-6
    except:
        return False
```

**两种模式使用完全相同的函数** ✅

## 两种模式对比

| 项目 | 完整模式 | 流式模式 |
|------|---------|---------|
| **对比逻辑** | ✅ 相同 | ✅ 相同 |
| **浮点数容忍** | 1e-6 | 1e-6 |
| **异常处理** | 记录errors.csv | 记录errors.csv |
| **输出报告** | analysis_report.csv | analysis_report.csv |
| **特征统计** | feature_stats.csv | feature_stats.csv |
| **中间文件** | ✅ 输出 | ❌ 不输出 |
| **内存占用** | 高 | 低（降低80-90%） |

## 输出文件

### 完整模式 (`output_intermediate_files: true`)

### 流式模式 (`output_intermediate_files: false`)
```

✅ feature_stats.csv         # 特征统计（相同）
✅ errors.csv                # 错误记录（相同）
```

## 使用建议

### 使用完整模式
- 数据量小（< 1万行）
- 需要查看接口原始数据
- 需要调试

### 使用流式模式
- 数据量大（> 1万行）
- 只需要对比报告
- 内存有限

---

## 配置

```json
{
  "output_config": {
    "output_intermediate_files": false  // false=流式模式
  }
}
```
