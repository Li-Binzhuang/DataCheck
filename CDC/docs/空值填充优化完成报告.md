## 🎯 修复原则

### 1. 统一空值标识

所有空值统一使用 -999 标识，包括：
- 数据缺失
- 无数据情况
- 计算结果为空

### 2. 区分"无数据"和"数值为0"

- **无数据**: 使用 -999
- **数值为0**: 保持 0

### 3. 保留数据清洗逻辑

数据质量检查中的 `= np.nan` 保持不变，用于标记异常数据，后续统一处理。

### 4. 添加修改注释

每处修改都添加注释：
- `# zlf update: 空值填充为-999`
- `# zlf update: 无数据时填充-999`
